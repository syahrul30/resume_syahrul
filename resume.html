mempelajari git

perintah dalam git:

  • git add = menambahkan file baru
  • git commit = menyimpan perubahan
  • git status = mengetahui status dari lokal repo
  • git push = mengirimkan perubahan file ke remote repo
  • git clone = mengunduh kode dari repo orang lain
  • git checkout = pindah branch
  • git branch -b = membuat branch baru
  • git fetch = update
  • git pull = mengambil commit terbaru dari remote repo
